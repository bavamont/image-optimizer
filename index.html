Help & Documentation

Learn how to use Image Optimizer effectively

Getting Started

1. Select your images by dragging and dropping or clicking to browse
2. Choose output location: same folder as source or custom folder
3. Configure processing options in Quick Settings
4. Click "Start Processing" to optimize your images
5. View results and open the output folder when complete

Image Combiner Getting Started

1. Add images or PDF files to the combiner
2. Choose alignment: vertical (stack) or horizontal (side by side)
3. Set maximum dimensions and spacing between images
4. Configure output format and quality settings
5. Click "Combine Images" to create your combined image

Supported Formats

Input Formats:

JPEG PNG WebP TIFF GIF AVIF SVG PDF

Output Formats:

JPEG PNG WebP TIFF AVIF

PDF Support:

PDF files are converted page-by-page into images for processing

Format Recommendations:

WebP and AVIF provide superior compression with excellent quality
PNG is ideal for images requiring transparency preservation
TIFF offers lossless compression for archival purposes
JPEG provides universal compatibility with good compression
SVG and GIF are supported as input formats only

Troubleshooting

Out of Memory:

Process fewer images at once or resize large images before processing

Unsupported Format:

Ensure input files are in supported formats (JPEG, PNG, WebP, TIFF, GIF, AVIF, SVG, PDF)

Permission Denied:

Check file and folder permissions, run as administrator if necessary

Insufficient Disk Space:

Free up disk space in the output directory before processing

Performance Tips:

Process images in smaller batches for better memory management. Use WebP or AVIF formats for optimal file size reduction. Disable watermarks when not needed to improve processing speed. Choose appropriate quality levels (80-95 for photos, 60-80 for web). Consider resizing large images to reduce processing time.
