Help & Documentation

Learn how to use Image Optimizer effectively

Getting Started

  1. Drag and drop images into the dropzone or click to browse files
  2. Choose output location: same folder as source or custom folder
  3. Configure processing options in Quick Settings (format, quality, resize, watermark, batch rename)
  4. Click "Start Processing" to optimize your images
  5. View results and open the output folder when complete

Key Features:

  • Batch processing of multiple images
  • Format conversion (JPEG, PNG, WebP, TIFF, AVIF)
  • Quality adjustment and compression
  • Image resizing with multiple fit modes
  • Watermark overlay with position and opacity control
  • Batch rename with patterns and counters
  • Preserve folder structure option

Supported Formats

Input Formats:

JPEG PNG WebP TIFF GIF AVIF SVG

Output Formats:

JPEG PNG WebP TIFF AVIF

Format Notes:

  • WebP and AVIF offer better compression than JPEG
  • PNG is best for images with transparency
  • TIFF is ideal for high-quality archival images
  • SVG and GIF are read-only formats

Settings Guide

Output Folder:

Choose to save processed images in the same folder as originals or select a custom output folder. Enable "Create subfolder" to organize processed images.

Resize Options:

Resize images to specific dimensions. Choose fit mode: Cover (crop to fit), Contain (fit inside), Fill (stretch), Inside (shrink only), or Outside (enlarge only).

Watermark:

Add a watermark image to all processed images. PNG files with transparency work best. Adjust position, opacity, and size as needed.

Batch Rename:

Rename files using patterns. Use {counter} for numbers, {date} for current date, {time} for current time, and {original} for original filename.
